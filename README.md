# Лабораторная работа

Реализована программа на C++ с базовым классом `Queue` и тремя дочерними классами:
- `Queue1Public` (`public` наследование),
- `Queue1Protected` (`protected` наследование),
- `Queue1Private` (`private` наследование).

## Что реализовано
- Инициализация очереди.
- Добавление элемента в очередь.
- Извлечение элемента из очереди.
- Глубокое копирование очереди (`createCopy`).
- Слияние двух очередей (`merge`) с сохранением порядка.
- Пользовательское меню (пункты 1–7 по заданию).
- Отдельный `main.cpp`.
- Разделение на `.h` и `.cpp`.
- Динамическая память для элементов очереди и корректное удаление.

> Примечание: в данной реализации для класса `Queue1*` вычисляемое значение —
> **количество положительных элементов** в очереди.

## Структура
- `Queue.h`, `Queue.cpp` — базовый класс очереди.
- `Queue1Public.h`, `Queue1Public.cpp` — дочерний класс (`public`).
- `Queue1Protected.h`, `Queue1Protected.cpp` — дочерний класс (`protected`).
- `Queue1Private.h`, `Queue1Private.cpp` — дочерний класс (`private`).
- `main.cpp` — точка входа и меню.

## Сборка
```bash
g++ -std=c++17 -Wall -Wextra -pedantic *.cpp -o queue_app
```

## Запуск
```bash
./queue_app
```
